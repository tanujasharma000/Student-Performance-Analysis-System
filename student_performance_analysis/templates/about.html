📌 Project Overview

This is a Student Performance Analysis System built using Flask and Python. It analyzes student marks from CSV files and generates automated insights, visual graphs, and email alerts for weak students.

⚙️ How It Works

  • 1. User uploads CSV file containing student data
  • 2. Backend reads file using Pandas
  • 3. Data is analyzed (average, pass/fail, categories)
  • 4. Session stores processed results
  • 5. Dashboard displays graphs & tables
  • 6. Emails are sent to weak students automatically

🚀 Features

  • Dynamic subject detection (no hardcoding)
  • Automatic performance analysis
  • Pass vs Fail visualization
  • Subject-wise average graphs
  • Attendance tracking
  • Performance categorization
  • Email alert system
  • Downloadable report

🧠 Analysis Logic

  • Average marks calculated per student
  • Weak students: average < 40
  • Excellent: ≥ 80
  • Good: 60–79
  • Average: 40–59

📊 Graphs Used

  • 📈 Subject Average (Bar Graph)
  • 🥧 Pass vs Fail (Pie Chart)
  • 📊 Performance Categories
  • 📉 Attendance Distribution

📩 Email System

Students with low performance are automatically identified and email alerts are sent using Flask SMTP integration.

🛠️ Tech Stack

  • Python (Pandas, Flask)
  • Matplotlib
  • HTML, CSS
  • Jinja Templates
  • SMTP Email System

🎯 Purpose

To help teachers analyze student performance easily, identify weak students, and automate reporting & alerts.
