# 📊 Student Performance Analysis System

A Python-based system to analyze student marks, visualize performance, and send automated email alerts to weak students.

## 📖 About the Project

Educational institutions generate large amounts of marks data, which is difficult to analyze manually. This project automates the analysis of student performance using Python, provides visual insights, and helps identify weak students. It also sends automatic email notifications to those students to help them improve.

---

## ❗ Problem Statement

Manual performance analysis is:
- Time-consuming and effort-intensive
- Lacking clear visualization
- Unable to quickly identify weak students
- Without automatic communication to students

---

## ✅ Proposed Solution

The system:
- Automatically calculates totals, averages, and percentages
- Generates visual graphs for better understanding
- Identifies weak students based on performance criteria
- Sends automated email alerts to weak students

---

## 🚀 Features

- 📁 Read student data from CSV files
- 📊 Calculate total marks, average, and percentage
- 📉 Visualize performance using graphs (Matplotlib)
- 🔍 Identify weak students (configurable threshold)
- ✉️ Send automated email alerts via SMTP
- 🌐 Web interface using Flask

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| Python | Core programming language |
| Pandas | Data processing & analysis |
| NumPy | Numerical operations |
| Matplotlib | Data visualization |
| Flask | Web interface & routing |

---

## 🔁 Workflow
Load CSV → Process Data → Calculate Metrics → Generate Graphs → Identify Weak Students → Send Email Alerts → Web Output
